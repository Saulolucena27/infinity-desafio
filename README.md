# Sistema de Controle de Estoque

Sistema de controle de estoque desenvolvido em Python com MySQL.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Edição de produtos
- Exclusão de produtos
- Registro de vendas
- Listagem de vendas
- Edição de vendas
- Exclusão de vendas

## Requisitos

- Python 3.x
- MySQL Server
- mysql-connector-python
